Project - GoCAN

Hey everyone, welcome to the first post here. One of the projects I’ve been working on recently is what I called “GoCAN". It’s an implementation of a Content Addressable Network, described here, which I’ve been working on with a friend of mine. What is a CAN? A Content Addressable Network is a network of systems which store a set of content-addressable data. What that means is the data in the network is stored in a distributed hash table, and the data is partitioned between different systems in the network....
